ARIA Live Regions Demo

1. role="alert" Example

Alerts are assertive by default and will interrupt the user immediately.

2. aria-live="polite" Example

Polite announcements wait for the user to finish their current task before announcing.

3. aria-live="assertive" Example

Assertive announcements interrupt the user immediately, similar to alerts.

4. aria-live="off" Example

This region is turned off and won't announce changes.
